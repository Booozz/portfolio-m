# Site - Project: Portfolio

`Node 12.16.1` `NPM 6.14.1` `PHP 7.3.15` `Composer 1.9.3`

![WOW.](https://cldup.com/JD_k_tMHGF.gif "WOW.")


## Setup
```
npm install
composer install
```

## Run
```
npm run start
```

## Build
```
npm run build
npm run build.preview
```

## Deploy
```
npm run deploy
npm run deploy.preview
```

## Debug
```
npm run start.debug
npm run build.debug
```
