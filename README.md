# Site - Project: Portfolio

`Node 14.16.1` `NPM 7.12.1` `Composer 2.0.14` `PHP 7.4.19` `WinSCP 5.17.10` `PowerShell 7.1.3`

## Setup
```
npm install
composer install
```

## Run
```
npm run start
```

## Build
```
npm run build
npm run build.preview
```

## Deploy
```
npm run deploy
npm run deploy.preview
```

## Debug
```
npm run start.debug
npm run build.debug
```
