# Site - Project: Portfolio

`Node 13.14.0` `NPM 6.14.8` `PHP 7.4.11` `Composer 1.10.15`

![WOW.](https://cldup.com/JD_k_tMHGF.gif "WOW.")


## Setup
```
npm install
composer install
```

## Run
```
npm run start
```

```
npm run build
npm run build.preview
```

## Deploy
```
npm run deploy
npm run deploy.preview
```

## Debug

```
npm run start.debug

```
