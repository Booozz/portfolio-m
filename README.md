# Site - Project: Portfolio

`Node 14.17.0` `NPM 7.16.0` `Composer 2.1.1` `PHP 7.4.20` `WinSCP 5.17.10` `PowerShell 7.1.3`

## Setup
```
npm install
composer install
```

## Run
```
npm run start
```

## Build
```
npm run build
npm run build.preview
```

## Deploy
```
npm run deploy
npm run deploy.preview
```

## Debug
```
npm run start.debug
npm run build.debug
```
